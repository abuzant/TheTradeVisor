# User Registration Guide

## Overview
This guide explains how to create and configure your TheTradeVisor account.

## Registration Process

### 1. Visit Registration Page
Navigate to `https://thetradevisor.com/register`

### 2. Fill Registration Form
- **Email**: Use a valid email address
- **Password**: Minimum 8 characters
- **Confirm Password**: Re-enter password
- **Name**: Your display name
- **Country**: Select your country from dropdown

### 3. Email Verification
After registration:
1. Check your email for verification link
2. Click the verification link
3. Your account will be activated

## Account Types

### Standard Account
- Full analytics access
- Up to 5 trading accounts
- Public profile (optional)
- Basic support

### Enterprise Account
- Unlimited trading accounts
- Advanced analytics
- White-label options
- Priority support
- API access

## Profile Configuration

### 1. Basic Information
- Update your name and email
- Set your timezone
- Configure currency display

### 2. Security Settings
- Enable two-factor authentication
- Set up API keys
- Configure session timeout

### 3. Notification Preferences
- Email notifications
- In-app alerts
- Mobile push notifications

## Troubleshooting

### Email Not Received
- Check spam folder
- Verify email address
- Request resend from login page

### Registration Errors
- Ensure all fields are filled
- Check password requirements
- Verify email format
