# User Guide: Export & Filter Features

## Quick Start Guide

### How to Filter Data by Date Range

1. **Navigate to any data page** (Trades, Symbol Analysis)
2. **Look for the date filter section** at the top of the page
3. **Select your dates:**
   - **Start Date:** Choose the beginning of your date range
   - **End Date:** Choose the end of your date range
   - Leave either blank to have an open-ended range
4. **Click "Filter"** to apply
5. **Click "Clear"** to reset and see all data

### How to Export Your Trading Data

#### Export All Trades
1. Go to **"My Trades"** page (`/trades`)
2. **(Optional)** Apply date range or other filters
3. Click **"Export CSV"** for spreadsheet format
4. **OR** Click **"Export PDF"** for printable report
5. File downloads automatically to your device

#### Export Symbol-Specific Data
1. Go to any symbol page (e.g., `/trades/symbol/XAUUSD`)
2. **(Optional)** Apply date range filter
3. Click **"Export CSV"**
4. Get all trades for that specific symbol

#### Export Dashboard Summary
1. Go to **"Dashboard"** page
2. Click **"Export Summary"** in the top right
3. Receive a CSV with all your accounts and their statistics

### Before Deleting Your Account

**IMPORTANT:** Download your data before deletion!

1. Go to **"Profile"** page
2. Scroll to **"Delete Account"** section
3. Look for the blue box: **"Download Your Data"**
4. Click **"Download All My Data (CSV)"**
5. Save the file securely
6. This file contains:
   - Your user information
   - All trading accounts
   - Complete trading history
   - All transactions

## File Formats Explained

### CSV Files (Spreadsheet)
- **Opens in:** Excel, Google Sheets, Numbers
- **Best for:** Data analysis, calculations, filtering
- **Contains:** Raw data in table format
- **Encoding:** UTF-8 (supports all characters)

### PDF Files (Report)
- **Opens in:** Any PDF reader
- **Best for:** Printing, sharing, archiving
- **Contains:** Formatted report with styling
- **Features:** Color-coded profits, professional layout

### Complete Account Data Export

## Tips & Tricks

### 📊 Analyzing Your Data
- Export to CSV and open in Excel/Google Sheets
- Use pivot tables for deeper analysis
- Create your own charts and graphs
- Compare different time periods

### 📅 Date Range Examples
- **Last Month:** Set start date to first day of last month, end date to last day
- **This Year:** Set start date to January 1st, leave end date blank
- **Specific Week:** Set both start and end dates 7 days apart
- **Single Day:** Set same date for both start and end

### 💾 Organizing Your Exports
- Files are automatically named with timestamps
- Example: `trades_2025-11-07_120530.csv`
- Easy to organize chronologically
- No confusion about which export is which

### 🔒 Data Privacy
- Only YOU can export YOUR data
- Exports are generated on-demand
- No data is stored on our servers after download
- Files are downloaded directly to your device

## Troubleshooting

### Export button not working?
- Check your internet connection
- Try refreshing the page
- Clear your browser cache
- Contact support if issue persists

### CSV file looks strange?
- Make sure you're opening with a spreadsheet app
- If characters look wrong, the app may not support UTF-8
- Try Google Sheets (best UTF-8 support)

### PDF not downloading?
- Check your browser's download settings
- Look in your Downloads folder
- Disable pop-up blockers temporarily
- Try a different browser

### Date filter not applying?
- Make sure you clicked "Filter" button
- Check that dates are in correct format
- End date must be after start date
- Try clicking "Clear" and re-applying

## Keyboard Shortcuts

- **Enter:** Submit filter (when in date field)
