# Dashboard Overview Guide

## Overview
This guide provides a comprehensive tour of TheTradeVisor dashboard interface.

## Main Dashboard Layout

### 1. Header Navigation
- **Logo**: Returns to main dashboard
- **Analytics**: Detailed analytics page
- **Accounts**: Manage trading accounts
- **Public Profile**: View your public profile
- **Settings**: Account and system settings
- **Logout**: Sign out of account

### 2. Summary Cards
At the top of the dashboard, you'll find key metrics:
- **Total Balance**: Combined balance across all accounts
- **Total Profit/Loss**: Overall performance
- **Open Positions**: Number of active trades
- **Total Trades**: Historical trade count

### 3. Account Selector
- Dropdown to switch between connected accounts
- Shows account name, broker, and balance
- "All Accounts" view for combined analytics

## Main Sections

### Performance Section
- **Equity Curve**: Visual representation of account balance over time
- **Profit/Loss Chart**: Daily/weekly/monthly P&L
- **Win Rate**: Percentage of profitable trades
- **Average Win/Loss**: Typical trade outcomes

### Recent Activity
- **Last 10 Trades**: Recent trade history
- **Open Positions**: Currently active trades
- **Pending Orders**: Orders waiting to execute
- **Account Events**: Deposits, withdrawals, and other activities

### Analytics Overview
- **Best Performing Symbols**: Top traded instruments
- **Trading Volume**: Lot sizes and frequency
- **Drawdown**: Current and historical drawdown
- **Risk Metrics**: Account risk statistics

## Interactive Features

### Time Period Selector
Choose from:
- Last 7 days
- Last 30 days
- Last 3 months
- Last 6 months
- Last year
- Custom date range

### Chart Controls
- **Zoom**: Click and drag on charts
- **Pan**: Click and drag to move view
- **Export**: Download charts as images
- **Full Screen**: Expand chart view

### Data Tables
- **Sort**: Click column headers
- **Filter**: Search and filter data
- **Export**: Download as CSV/Excel
- **Pagination**: Navigate large datasets

## Account Management

### Add New Account
1. Click "Add Account" button
2. Fill account details
3. Install Expert Advisor
4. Verify connection

### Account Status
- **Connected**: Green indicator
- **Syncing**: Yellow indicator
- **Disconnected**: Red indicator
- **Error**: Red with error message

### Account Actions
- **Edit**: Update account details
- **Remove**: Disconnect account
- **Sync**: Manual data sync
- **Refresh**: Update account status

## Customization Options

### Dashboard Layout
- Drag and drop cards to rearrange
- Minimize/maximize sections
- Save layout preferences

### Display Settings
- **Currency**: Choose display currency
- **Timezone**: Set local timezone
- **Date Format**: MM/DD/YYYY or DD/MM/YYYY
- **Number Format**: Decimal places

### Color Themes
- Light theme (default)

- Custom colors

## Keyboard Shortcuts

### Navigation
- `Ctrl + 1`: Dashboard
- `Ctrl + 2`: Analytics
- `Ctrl + 3`: Accounts
- `Ctrl + 4`: Settings

### Actions
- `Ctrl + R`: Refresh data
- `Ctrl + E`: Export data
- `Ctrl + P`: Print view
- `Esc`: Close modals

## Performance Tips

### Optimize Loading
- Use appropriate time ranges
- Limit displayed accounts
- Clear cache regularly
- Use stable internet connection

### Large Datasets
- Export for detailed analysis
- Use filters to narrow data
- Consider summary views
- Archive old data

## Troubleshooting

### Common Issues
- **Data Not Loading**: Check connection, refresh page
- **Charts Not Displaying**: Update browser, clear cache
- **Slow Performance**: Reduce time range, close other tabs
- **Login Issues**: Clear cookies, reset password

### Error Messages
- **Connection Failed**: Check EA status
- **Data Sync Error**: Reconnect account
- **Permission Denied**: Check account settings
- **Rate Limited**: Wait, then retry
