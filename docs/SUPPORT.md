## Quick Fixes

### Password Reset
1. Go to Login page
2. Click "Forgot Password"
3. Enter your email
4. Check email for reset link
5. Create new password

### EA Connection
1. Verify EA is installed correctly
2. Check API key in EA settings
3. Ensure AutoTrading is enabled
4. Restart MT4/MT5 terminal
5. Check Experts tab for errors

### Data Sync
1. Check account status in Settings
2. Verify internet connection
3. Refresh account manually
4. Check time synchronization
5. Reconnect if necessary

### Clear Browser Cache
1. Press Ctrl+Shift+Delete
2. Select "All time"
3. Check "Cached images and files"
4. Click "Clear data"
5. Refresh page
